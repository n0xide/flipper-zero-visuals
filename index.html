Flipper Zero curriculum — visual companion

Interactive HTML visualizations for the topics that are hardest to grok from text alone. Modulation waveforms, MIFARE sector maps, GPIO pinouts, WiFi handshake sequences, attack-chain timelines — every visual that pays off across multiple study sessions lives here.

How to use this

When a phase markdown sends you to a visual (e.g., "see visuals/phase-2-subghz.html#modulation"), open it in a browser. Click and drag — every visual is interactive. The visuals are deliberately complementary, not replacements: read the phase markdown for theory; use the visuals when your brain wants a picture.

Why nine phases, not all ten

Each phase visualization burns ~1500 lines of HTML. To stay valuable, the visuals target topics that are genuinely hard to grok from text. Only Phase 0 stays text-only — pure policy that a diagram doesn't help.
